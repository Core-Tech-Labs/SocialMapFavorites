### IMPORTANT: THIS PACKAGE IS CURRENTLY IN DEVELOPEMNT.

## Install

Via Composer

``` bash
$ composer require ctl/socialmapfavorites
```

## Usage

``` bash
$ php artisan vendor:publish
```

## Testing

``` bash
$ phpunit
```
